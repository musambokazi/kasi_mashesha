# React Native Authentication App

This is a simple React Native app that allows users to register and login as either a business owner or a driver.

## Features
* User registration with name, email, password, and role (business owner or driver)
* User login with email, password, and role
* Validation for empty fields
* Simple and clean UI

## Installation
1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npx react-native start` to start the app

## Usage
1. Open the app on your device or emulator
2. Register or login as a business owner or driver
3. Fill in the required fields and submit

## Code Structure
* `Register.js`: Registration screen
* `Login.js`: Login screen
* `Navigation.js`: Navigation setup
* `App.js`: Main app component

## Dependencies
* `@react-navigation/native`
* `@react-navigation/stack`
* `react-native-screens`
* `react-native-safe-area-context`
